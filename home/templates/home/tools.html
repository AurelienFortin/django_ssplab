Outils et codes

Le SSP Lab met à disposition, dès que c'est possible, le code issu de ses expérimentations, ainsi que des outils qui peuvent etre utilisés largement au sein du SSP.

Pour accéder au code des principales expérimentations, les repos de référence sont situés :

  • Sur le GitLab de l'Insee
  • Plus rarement, sur le repo du git lab de la plateforme du datalab

En termes d'outils, le SSP Lab tente de développer des API pour que les résultats des expérimentations. A ce stade, deux API sont prêtes ou en cours d'élaboration :

  • (disponible) une API qui permet, à partir du libellé d'un produit, d'obtenir sa famille classifiée. Cette API a été permise par l'exploitation de données de caisse.
  • (en cours) une API qui, à partir d'un champ texte, propose un SIRET possible
